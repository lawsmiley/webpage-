## Required Images

### 1. **Emblem Logo** (`/public/emblem.png`)
- **Size:** Recommended 512x512px or larger
- **Format:** PNG with transparency
- **Usage:** Appears in navbar and hero section
- **Placeholder:** You can use any square logo/image for now

### 2. **Project Images** (`/public/projects/`)
Create a `projects` folder in `public/` and add:

- **`nebula.png`** - Design system project thumbnail
- **`aurora.png`** - E-commerce project thumbnail  
- **`voyager.png`** - Travel app project thumbnail

**Recommended specs:**
- **Size:** 800x600px or larger
- **Format:** PNG or JPG
- **Aspect ratio:** 4:3 or 16:9

## Quick Setup

1. **Add your emblem:**
   ```bash
   # Place your logo at:
   public/emblem.png
   ```

2. **Create projects folder:**
   ```bash
   mkdir public/projects
   ```

3. **Add project images:**
   ```bash
   # Add these files:
   public/projects/nebula.png
   public/projects/aurora.png
   public/projects/voyager.png
   ```

## Temporary Placeholders

If you don't have images yet, you can:
- Use placeholder services like `https://via.placeholder.com/512` for emblem
- Use `https://via.placeholder.com/800x600` for projects
- Or create simple colored divs as placeholders

The site will work without images, but they'll show broken image icons until you add them.
